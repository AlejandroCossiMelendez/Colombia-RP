# Scoreboard Custom - Instrucciones

## Para activar el scoreboard personalizado:

1. **Reinicia el servidor completo** o ejecuta en la consola del servidor:
   ```
   restart scoreboard_custom
   ```

2. Si no funciona, det√©n el scoreboard original:
   ```
   stop scoreboard
   ```
   
   Y luego inicia el personalizado:
   ```
   start scoreboard_custom
   ```

## Nota importante:
- Solo reiniciar `login` NO cambia el scoreboard
- El scoreboard es un recurso separado que debe reiniciarse por separado
- Presiona TAB en el juego para ver el nuevo scoreboard
