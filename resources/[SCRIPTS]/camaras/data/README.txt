CARPETA DE RECURSOS VISUALES - SISTEMA CCTV v2.0
================================================

Esta carpeta contiene recursos visuales opcionales para el sistema de cámaras:

ARCHIVOS OPCIONALES:
- Rectangle 1.png       - Fondo del panel de control
- flechaArriba.png      - Botón cambiar cámara
- flechaIzquierda.png   - Botón rotar izquierda  
- flechaDerecha.png     - Botón rotar derecha
- font.ttf              - Fuente personalizada
- medium.ttf            - Fuente media

IMPORTANTE:
- El sistema funciona perfectamente SIN estos archivos
- Si los archivos no existen, se usan elementos visuales por defecto
- Para usar imágenes personalizadas, coloca los archivos aquí y descomenta las líneas correspondientes en meta.xml

RESOLUCIONES RECOMENDADAS:
- Botones: 64x64 píxeles (PNG con transparencia)
- Panel: 400x350 píxeles (PNG con transparencia)

El sistema es completamente funcional sin estos recursos visuales.
